# 투자 전략 페이지 스크롤 스토리텔링 디자인

## 목적
운영자의 투자 철학과 전략을 방문자에게 소개하는 페이지.
현재 텍스트만 있는 단일 페이지를 스크롤 스토리텔링 형태로 개편.

## 페르소나 (about 페이지 기반)
- 40대 초반 IT 직장인, 겁쟁이 투자자
- 분석 중심, 인과관계 정리 마니아
- 한국장 오래 함, 미국장은 자동화
- 말투: 솔직하고 자기객관화된 톤

## 콘텐츠 구조 (4 챕터)

### Chapter 1: 투자 철학 — 5가지 가치
- 테마(매크로), 펀더멘탈, 추세, 거래량, 시간
- 계층 필터로 구조화:
  1. 테마 — 거시 방향이 맞는가? (1차 필터)
  2. 추세 — 기술적 타이밍이 왔는가? (2차 필터)
  3. 펀더멘탈 — 안전판이 있는가? (3차 필터)
  4. 거래량 — 시장이 동의하는가? (확인 신호)
  5. 시간 — 기회비용을 감당할 수 있는가? (최종 점검)

### Chapter 2: 거시 판단 워크플로우
- 금리/통화정책 체크
- 환율/달러 방향성
- 유동성/자금 흐름 (외국인, 기관, 고객예탁금)
- 정부정책 기대값 평가
- 이 4단계를 거쳐 "주도 테마의 현재 위치" 추측

### Chapter 3: 종목 선정 기준
- 기술적 진입 조건: MACD 주봉 양수 AND 스토캐스틱 슬로우 주봉 양수
- 테마 체크: 시장 관심 테마인가
- 펀더멘탈 체크: 나쁘지 않은가
- 저평가 종목: 저평가인 이유를 꼼꼼히 조사

### Chapter 4: 보유종목을 대하는 태도
- 변동 시 인과관계 분석 (공부 → 회귀분석 → 같은 실수 확률 낮춤)
- fast fail → not fallback (실패 인정)
- 기다리는 시간도 기대수익
- 수익 실현해야 내 돈

## UI 설계 — 스크롤 스토리텔링

### 레이아웃
- single page, 4개 full-viewport 챕터
- 각 챕터 fade-in/slide-up on scroll (IntersectionObserver)
- 진행률 인디케이터 (좌측 또는 상단)
