## 한줄 요약

**테이퍼링 = 돈 풀기(QE)를 서서히 줄이는 것 (수도꼭지 잠그기 시작)**

양적완화 → **테이퍼링** → 양적긴축(QT) 순서로 진행됩니다.

## 쉽게 이해하기

- **양적완화(QE)** = 수도꼭지를 활짝 틀어 돈을 쏟아붓기
- **테이퍼링** = 수도꼭지를 조금씩 잠그기 (아직 물은 나옴)
- **양적긴축(QT)** = 수도꼭지를 완전히 잠그고 물을 빼기

## 왜 중요한가

- 테이퍼링 발표 = "앞으로 돈이 줄어든다" 시그널
- 2013년 **테이퍼 탠트럼**: 버냉키 의장이 테이퍼링 언급만 했는데 채권시장 폭락
- 시장은 실제 긴축보다 **긴축 예고**에 더 민감하게 반응

## 타임라인 (전형적)

```
경기침체 → QE 시작 → 경기회복 → 테이퍼링 → QE 종료 → 금리인상 → QT
```
