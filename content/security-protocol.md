## 보안 운영 원칙

- 사이트는 정적 배포(Hugo) 기반으로 운영되며, 서버 측 사용자 세션 정보를 저장하지 않습니다.
- 운영 및 배포 과정에서 최소 권한 원칙을 적용합니다.
- 외부 데이터 소스는 공개 API 또는 공개 피드를 우선 사용합니다.

## 취약점 제보

- 포함 정보: 영향 범위, 재현 단계, 예상 위험도
