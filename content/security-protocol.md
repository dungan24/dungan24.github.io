## 보안 운영 원칙

- 사이트는 정적 배포(Hugo) 기반으로 운영되며 서버 세션 저장을 최소화합니다.
- 운영/배포 자동화에는 최소 권한 원칙을 적용합니다.
- 외부 데이터는 공개 API/피드 기반으로 수집합니다.
