`assets/css/custom/layout-overrides.css`의 `/* Mobile Bottom Nav (T-402) */` 주석(114행) **바로 위**에 아래 CSS 블록을 추가한다. 모바일 바텀 네비 섹션과 구분되도록 빈 줄로 분리.

```css
/* === NAV-01: Frosted Glass Sticky Header === */
/* Blowfish fixed.html의 #menu-blur Tailwind 클래스(bg-neutral/25, backdrop-blur-2xl) 오버라이드 */
/* opacity는 건드리지 않음 — background-blur.js가 scroll 위치에 따라 제어 */
#menu-blur {
  background: rgba(10, 10, 26, 0.78) !important;
  -webkit-backdrop-filter: var(--mp-glass-blur) !important;
  backdrop-filter: var(--mp-glass-blur) !important;
  border-bottom: 1px solid rgba(124, 58, 237, 0.12);
  box-shadow: 0 4px 24px -4px rgba(0, 0, 0, 0.5);
}

/* === NAV-02: Light Mode Header === */
:root:not(.dark) #menu-blur {
  background: rgba(255, 255, 255, 0.88) !important;
  border-bottom-color: rgba(124, 58, 237, 0.08);
  box-shadow: 0 4px 16px -4px rgba(0, 0, 0, 0.06);
}

/* === NAV-02: Dark Mode Nav Link Colors === */
.dark .main-menu a,
.dark .main-menu span {
  color: rgba(var(--color-neutral-100), 1);
}
.dark .main-menu a:hover {
  color: rgba(var(--color-primary-400), 1);
}

/* === NAV-02: Light Mode Nav Link Colors === */
/* 흰 배경 위에서 텍스트 대비 보장: neutral-900 (#0A0A1A) */
:root:not(.dark) .main-menu a,
:root:not(.dark) .main-menu span {
  color: rgba(var(--color-neutral-900), 1);
}
:root:not(.dark) .main-menu a:hover {
  color: rgba(var(--color-primary-400), 1);
}
```

**핵심 주의사항:**

1. **`!important` 필수:** Blowfish Tailwind 컴파일 CSS의 `bg-neutral/25`, `backdrop-blur-2xl`를 오버라이드하려면 `!important` 필요. 이는 Phase 1 연구에서 검증된 패턴.

2. **`opacity` 절대 건드리지 말 것:** `background-blur.js`가 `#menu-blur`의 `opacity`를 `scroll / 300` 인라인 스타일로 제어. 페이지 최상단에서 투명 → 스크롤 내릴수록 불투명 전환이 의도된 동작.

3. **`backdrop-filter` 단축 속성으로 완전 교체:** `var(--mp-glass-blur)`가 `blur(14px) saturate(1.4)` 전체 함수 문자열이므로 Tailwind의 `--tw-backdrop-blur` 변수를 무효화. 이중 blur 방지됨 (Phase 1 FOUN-02와 동일 패턴).

4. **색상 변수 사용:** `rgba(var(--color-neutral-100), 1)` 패턴은 Blowfish/cyberpunk.css가 `--color-neutral-100: 226 232 240` 형태로 정의하므로 해당 RGB 값이 대입됨.

5. **Blowfish 테마 파일(`themes/blowfish/`) 절대 수정 금지:** CSS 오버라이드만으로 구현. 테마 업데이트 시 덮어씌워지는 것 방지.

6. **NAV-03 (모바일 반응형):** 기존 `#mp-mobile-bottom-nav` CSS(119-157행)가 이미 glass 스타일(`backdrop-filter: blur(16px)`, `rgba(10,10,26,0.9)`)을 가지고 있으므로 추가 수정 불필요. 이모지 → Phosphor 아이콘 교체는 Plan 01이 담당하며, `.mp-nav-icon { font-size: 1.2rem }` 규칙은 `<i>` 태그에도 그대로 상속됨. 라이트 모드 바텀 네비(159-165행)도 이미 적절한 대비를 제공.
